Notebooks to play with

Besides the GUI, you can also control the graph via notebook. The notebook cells have a built-in graph variable that you use to call addVertex, addEdge, removeVertex, and removeEdge, allowing you to script a graph to your liking. The cells support await and return (in fact they get executed as asynchronous functions). You can return strings, numbers, and html elements. For more functionality, see import(...) on MDN.
Below are a short handful of notebooks. Cube and Micrograd have shapes for you to play with, while Quickstart and Minimal Graph are about providing detailed instructions on how to use the notebook.

What can I do with it?

Using fetch(...) call, you can load data from a url and visualize it using the graph. The possibilities are endless, basically anything with arrows.
